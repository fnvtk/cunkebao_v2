# 自动点赞

## 功能概述

自动点赞功能可以根据设定的规则，自动为指定好友的朋友圈内容点赞，增强互动频率，提升好感度和活跃度。

## 功能入口

工作台 > 自动点赞

## 操作流程

1. 进入自动点赞页面
2. 点击"新建点赞任务"按钮
3. 设置任务基本信息（任务名称、执行周期等）
4. 选择执行设备
5. 选择点赞对象（可按标签、活跃度等筛选好友）
6. 设置点赞规则（内容类型、关键词、时间范围等）
7. 设置点赞频率和数量限制
8. 保存并启动任务

## 功能特点

- 智能筛选：根据内容类型、关键词等智能筛选要点赞的内容
- 时间控制：支持设置点赞的时间范围和频率
- 数量限制：可设置每日最大点赞数量，避免异常行为
- 标签分组：支持按标签选择点赞对象
- 优先级设置：可为重要客户设置更高的点赞优先级
- 执行报告：提供详细的点赞记录和统计数据

## 注意事项

- 单个设备每日点赞数量建议不超过100个
- 点赞行为应模拟真实用户习惯，避免在短时间内大量点赞
- 定期检查任务执行情况，及时调整策略
- 避免频繁修改点赞规则，可能导致行为异常

## 常见问题

**Q: 自动点赞会不会被微信检测到是机器行为？**
A: 系统已内置随机延时和智能筛选机制，模拟真实用户行为，但仍建议设置合理的点赞频率和数量，避免触发风控。

**Q: 如何设置更精准的点赞目标？**
A: 可以通过组合使用标签筛选、关键词筛选和内容类型筛选，例如只对"重要客户"标签的好友发布的包含"产品"关键词的图片类型朋友圈进行点赞。
